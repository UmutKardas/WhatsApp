<h1>WhatsApp Clone – SwiftUI Real-Time Messaging App</h1>

<h2>Overview</h2>
<p>
  <strong>WhatsApp Clone</strong> is a modern messaging application built with SwiftUI, designed to replicate the clean and familiar WhatsApp chat experience.
  The project includes both a chats list screen and a real-time messaging screen, styled to closely match WhatsApp’s UI behavior.
</p>
<p>
  With Firebase Firestore and Firebase Authentication, the app supports real-time messaging inside a scalable MVVM architecture powered by Combine and Swinject.
</p>

<hr />

<h2>Features</h2>
<ul>
  <li>WhatsApp-style chat interface</li>
  <li>Real-time messaging using Firebase Firestore</li>
  <li>User authentication with Firebase Auth</li>
  <li>Custom <strong>ChatBubble</strong> component for message UI</li>
  <li>Chats list screen</li>
  <li>Message detail screen</li>
  <li>Message timestamps</li>
  <li>Auto-scroll to the latest message</li>
  <li>Fully responsive SwiftUI layout</li>
  <li>Reactive data flow using Combine</li>
  <li>MVVM architecture with Swinject dependency injection</li>
</ul>

<hr />

<h2>Technologies Used</h2>
<ul>
  <li><strong>SwiftUI</strong></li>
  <li><strong>Combine</strong></li>
  <li><strong>MVVM Architecture</strong></li>
  <li><strong>Swinject</strong></li>
  <li><strong>Firebase Authentication</strong></li>
  <li><strong>Firebase Firestore</strong></li>
  <li><strong>Custom Component:</strong> ChatBubble</li>
</ul>

<h2>Installation</h2>
<ol>
  <li>Clone the repository</li>
  <li>Open the project folder in Xcode</li>
  <li>Update Swift Package Manager dependencies</li>
  <li>Add your <code>GoogleService-Info.plist</code> file</li>
  <li>Build and run on a simulator or physical device</li>
</ol>
